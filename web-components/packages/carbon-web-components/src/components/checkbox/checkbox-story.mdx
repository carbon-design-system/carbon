its label.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/web-components/es/components/checkbox/index.js';
```

<Description markdown={`${cdnJs({ components: ['checkbox'] })}`} />

### HTML

```html
<cds-checkbox label-text="Lorem Ipsum"></cds-checkbox>
```

## `<cds-checkbox>` attributes, properties and events

Unlike regular checkbox, `<cds-checkbox>` does _not_ fire `change` event. Please
use `cds-checkbox-changed` event instead.

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g.
`<cds-checkbox checked>`) and `false` means not setting the attribute (e.g.
`<cds-checkbox>` without `checked` attribute).
