Represents a link. Provided for convenience purpose, you can alternatively use
`<a class="cds--link">` with Carbon style in place in light DOM.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/web-components/es/components/list/index.js';
```

<Description markdown={`${cdnJs({ components: ['list'] })}`} />
<Description markdown={`${cdnCss()}`} />

### HTML

```html
<cds-link href="https://www.ibm.com"> Link </cds-link>
```

## Paired with icon

Import the desired icon and incorporate with the link text:

```javascript
import Download16 from '@carbon/web-components/es/icons/download/16';

function App() {
  return (
    html`<cds-link href="https://www.ibm.com"> Download ${Download16({ slot: 'icon' })}</cds-link>`
  );
}

```

## `<cds-link>` attributes and properties

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g.
`<cds-link disabled>`) and `false` means not setting the attribute (e.g.
`<cds-link>` without `disabled` attribute).
