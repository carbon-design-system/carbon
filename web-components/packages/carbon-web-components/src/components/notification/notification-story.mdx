## Inline notification

Inline notifications show up in task flows, to notify users of the status of an
action. They usually appear at the top of the primary content area.

```html
<bx-inline-notification
  style="min-width: 30rem; margin-bottom: .5rem"
  title="Notification title"
  subtitle="Subtitle text goes here.">
</bx-inline-notification>
```

## Toast notification

Toasts are a non-modal, time-based window elements used to display short
messages; they usually appear at the bottom of the screen and disappear after a
few seconds.

```html
<bx-toast-notification
  style="min-width: 30rem; margin-bottom: 0.5rem;"
  title="Notification title"
  subtitle="Subtitle text goes here.">
</bx-toast-notification>
```

## Themeing

In order to get the correct token value for the notification background, you
will need to apply the following in you `scss` to your themed class. For
example:

```css
@use '@carbon/styles/scss/vendor/@carbon/elements/scss/themes/mixins';
@use '@carbon/styles/scss/component-tokens';
@import 'carbon-components/src/components/notification/inline-notification';
@import 'carbon-components/src/components/notification/toast-notification';
.dds-theme-zone-g90 {
  @include carbon--theme($carbon--theme--g90, true) {
    @include emit-component-tokens($notification-colors);
  }
}
```
