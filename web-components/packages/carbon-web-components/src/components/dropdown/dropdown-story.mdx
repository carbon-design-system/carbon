content.

## Selection

When user attempts to select a dropdown item, `bx-dropdown-beingselected` event
fires on `<bx-dropdown>`, which has the following properties in the event
details:

| Property | Description                              |
| -------- | ---------------------------------------- |
| `item`   | The `<bx-dropdown-item>` being selected. |

`bx-dropdown-beingselected` bubbles and is cancelable. Canceling this event
means canceling change in selection.
