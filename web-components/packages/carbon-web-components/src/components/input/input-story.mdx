Text fields enable the user to interact with and input data. A single line field
is used when the input anticipated by the user is a single line of text as
opposed to a paragraph.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/web-components/es/components/input/index.js';
```

### HTML

```html
<bx-input>
  <span slot="label-text">Label text</span>
  <span slot="helper-text">Optional helper text</span>
  <span slot="validity-message">Something isn't right</span>
</bx-input>
```

## Form item

If the input is part of a form, make sure to import and wrap the input with
`<bx-form-item>`

```javascript
import '@carbon/web-components/es/components/form/index.js';
```

<Description markdown={`${cdnJs({ components: ['form'] })}`} />

```html
<bx-form-item>
  <bx-input>
    <span slot="label-text">Label text</span>
    <span slot="helper-text">Optional helper text</span>
    <span slot="validity-message">Something isn't right</span>
  </bx-input>
</bx-form-item>
```

## `<bx-input>` attributes and properties

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g.
`<bx-input required>`) and `false` means not setting the attribute (e.g.
`<bx-input>` without `required` attribute).
