Date and time pickers allow users to select a single or a range of dates and
times.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/carbon-web-components/es/components/date-picker/index.js';
```

<Description markdown={`${cdnJs({ components: ['date-picker'] })}`} />
<Description markdown={`${cdnCss()}`} />

### HTML

```html
<bx-date-picker>
  <bx-date-picker-input label-text="Date Picker label" placeholder="mm/dd/yyyy">
  </bx-date-picker-input>
</bx-date-picker>
```

## Single date picker

A single Date Picker consists of an input field and a calendar.

```html
<bx-date-picker>
  <bx-date-picker-input
    kind="single"
    label-text="Date Picker label"
    placeholder="mm/dd/yyyy"
  >
  </bx-date-picker-input>
</bx-date-picker>
```

## Range date picker

A range Date Picker consists of two input fields and a calendar.

```html
<bx-date-picker>
  <bx-date-picker-input
    kind="from"
    label-text="Date Picker label"
    placeholder="mm/dd/yyyy"
    size="lg"
  >
  </bx-date-picker-input>
  <bx-date-picker-input
    kind="to"
    label-text="Date Picker label"
    placeholder="mm/dd/yyyy"
    size="lg"
  >
  </bx-date-picker-input>
</bx-date-picker>
```
