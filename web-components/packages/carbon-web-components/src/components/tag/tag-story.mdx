Tags are used for items that need to be labeled, categorized, or organized using
keywords that describe them.

Multiple or single tags can be used to categorize items.

Use tags when content is mapped to multiple categories, and the user needs a way
to differentiate between them.

## Themeing

In order to get the correct token value for the tag background, you will need to apply the following in you `scss` to your themed class. For example:

```css
@import 'carbon-components/scss/globals/scss/vendor/@carbon/elements/scss/themes/mixins';
@import 'carbon-components/scss/globals/scss/component-tokens';
@import 'carbon-components/src/components/tag/tag';
.dds-theme-zone-g90 {
  @include carbon--theme($carbon--theme--g90, true) {
    @include emit-component-tokens($tag-colors);
  }
}
```
