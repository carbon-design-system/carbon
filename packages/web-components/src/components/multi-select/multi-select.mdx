## Overview

The `cds-multi-select` component is a variant of the `dropdown` component that
allows for multiple items to be selected. The `cds-multi-select` component stays
open while items are being selected.

<Canvas of={MultiSelectStories.Default} />

### With initial selected items

By using the `selected` attribute, `cds-multi-select-item` components can be
pre-selected.

<Canvas of={MultiSelectStories.WithInitialSelectedItems} />

### Filtering

Setting the `filterable` attribute to `true` allows for filtering the list of
selectable items to a subset.
