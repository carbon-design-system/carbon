Date and time pickers allow users to select a single or a range of dates and
times.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/web-components/es/components/date-picker/index.js';
```

<Markdown>{`${cdnCss()}`}</Markdown>

### HTML

```html
<cds-date-picker>
  <cds-date-picker-input
    label-text="Date Picker label"
    placeholder="mm/dd/yyyy">
  </cds-date-picker-input>
</cds-date-picker>
```

## Single date picker

A single Date Picker consists of an input field and a calendar.

```html
<cds-date-picker>
  <cds-date-picker-input
    kind="single"
    label-text="Date Picker label"
    placeholder="mm/dd/yyyy">
  </cds-date-picker-input>
</cds-date-picker>
```

## Range date picker

A range Date Picker consists of two input fields and a calendar.

```html
<cds-date-picker>
  <cds-date-picker-input
    kind="from"
    label-text="Date Picker label"
    placeholder="mm/dd/yyyy"
    size="lg">
  </cds-date-picker-input>
  <cds-date-picker-input
    kind="to"
    label-text="Date Picker label"
    placeholder="mm/dd/yyyy"
    size="lg">
  </cds-date-picker-input>
</cds-date-picker>
```


### Single

```html
<cds-date-picker-input-skeleton kind="single"></cds-date-picker-input-skeleton>
```

### Range

```html
<div>
  <cds-date-picker-input-skeleton kind="from"></cds-date-picker-input-skeleton>
  <cds-date-picker-input-skeleton kind="to"></cds-date-picker-input-skeleton>

```
