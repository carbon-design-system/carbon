Progress indicator is a visual representation of a users progress through a set
of steps. They guide the user through a number of steps in order to complete a
specified process.

Use progress indicators to keep the user on track when completing a specific
task. By dividing the end goal into smaller, sub-tasks, it increases the
percentage of completeness as each task is completed.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/web-components/es/components/progress-indicator/index.js';
```

### HTML

```html
<cds-progress-indicator>
  <cds-progress-step
    label-text="Label"
    secondary-label-text="Secondary label"
    state="invalid"></cds-progress-step>
  <cds-progress-step
    label-text="Label"
    secondary-label-text="Secondary label"
    state="complete"></cds-progress-step>
  <cds-progress-step
    label-text="Label"
    secondary-label-text="Secondary label"
    state="current"></cds-progress-step>
  <cds-progress-step
    disabled
    label-text="Label"
    secondary-label-text="Secondary label"></cds-progress-step>
  <cds-progress-step
    label-text="Label"
    secondary-label-text="Secondary label"></cds-progress-step>
</cds-progress-indicator>
```

## Skeleton

For the skeleton variation, utilize `<cds-progress-indicator-skeleton>` and
`<cds-progress-step-skeleton>`.

```html
<cds-progress-indicator-skeleton>
  <cds-progress-step-skeleton></cds-progress-step-skeleton>
  <cds-progress-step-skeleton></cds-progress-step-skeleton>
  <cds-progress-step-skeleton></cds-progress-step-skeleton>
</cds-progress-indicator-skeleton>
```
