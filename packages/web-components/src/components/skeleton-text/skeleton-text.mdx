## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/web-components/es/components/skeleton-text/index.js';
```

<Markdown>{`${cdnJs({ components: ['skeleton-text'] })}`}</Markdown>
<Markdown>{`${cdnCss()}`}</Markdown>

### HTML

```html
<cds-skeleton-text></cds-skeleton-text>
```

### Lines

To create multiple lines of skeleton text, set the `type` property and use
multiple instances of `<cds-skeleton-text>`.

```html
<cds-skeleton-text type="line"></cds-skeleton-text>
<cds-skeleton-text type="line"></cds-skeleton-text>
<cds-skeleton-text type="line"></cds-skeleton-text>
```
