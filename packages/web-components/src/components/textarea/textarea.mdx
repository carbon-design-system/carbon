Text inputs enable the user to interact with and input content and data. This
component can be used for long and short form entries.

Textarea is a multi-line variant of text input.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/web-components/es/components/textarea/index.js';
```

<Markdown>{`${cdnJs({ components: ['textarea'] })}`}</Markdown>
<Markdown>{`${cdnCss()}`}</Markdown>

### HTML

```html
<cds-textarea placeholder="Optional placeholder text" rows="4" cols="50">
  <span slot="label-text">Label text</span>
  <span slot="helper-text">Optional helper text</span>
  <span slot="validity-message">Something isn't right</span>
</cds-textarea>
```

## Form item

If the textarea is part of a form, make sure to import and wrap the textarea
with `<cds-form-item>`

```javascript
import '@carbon/web-components/es/components/form/index.js';
```

```html
<cds-form-item>
  <cds-textarea placeholder="Optional placeholder text" rows="4" cols="50">
    <span slot="label-text">Label text</span>
    <span slot="helper-text">Optional helper text</span>
    <span slot="validity-message">Something isn't right</span>
  </cds-textarea>
</cds-form-item>
```

## Skeleton

For the skeleton variation, utilize `<cds-textarea-skeleton>`.

```html
<cds-textarea-skeleton></cds-textarea-skeleton>
```

## `<cds-textarea>` attributes and properties

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g.
`<cds-textarea autofocus>`) and `false` means not setting the attribute (e.g.
`<cds-textarea>` without `autofocus` attribute).
