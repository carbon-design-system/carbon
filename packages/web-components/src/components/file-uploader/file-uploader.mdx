File uploaders allow users to select one or more files to upload to a specific
location.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/web-components/es/components/file-uploader/index.js';
```

<Markdown>{`${cdnJs({ components: ['file-uploader'] })}`}</Markdown>
<Markdown>{`${cdnCss()}`}</Markdown>

### HTML

```html
<cds-file-uploader
  label-description="Only .jpg and .png files. 500kb max file size"
  label-title="Account photo"
>
  <cds-file-uploader-button accept="image/jpeg image/png" multiple>
    Drag and drop files here or click to upload
  </cds-file-drop-container>
</cds-file-uploader>
```
