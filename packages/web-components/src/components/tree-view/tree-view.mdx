## Overview

TreeView is a component that allows users to navigate through a hierarchy of
items. It is used to display nested lists of items that can be expanded or
collapsed to show and hide additional items.

<Canvas of={TreeViewStories.Default} />

## With Controlled Expansion

The `is-expanded` prop can be used to determine if an individual tree node is
expanded or collapsed.

> The `controlled` attribute used here is for this example only, and is not part
> of the component API. Any valid DOM selector can be used.
