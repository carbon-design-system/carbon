## Default Icons

To enable the default icons in Tiles, enable the feature flag in Javascript

```scss
<FeatureFlags flags={{ 'enable-v12-tile-default-icons': true }}>
  <ClickableTile
    id="clickable-tile-1"
    href="https://www.carbondesignsystem.com/">
    Clickable Tile
  </ClickableTile>
</FeatureFlags>
```
