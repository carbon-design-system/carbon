## Overview

`SidePanel` keeps users in-context of a page while performing tasks like
navigating, editing, viewing details, or configuring something new. It can slide
over the page content (default) or slide in and push the page content aside.

Open and close the panel by toggling the `open` prop. Provide an
`onRequestClose` handler to update the state when the user clicks the close
button or presses Escape.

```jsx
const [open, setOpen] = useState(false);

<SidePanel
  open={open}
  onRequestClose={() => setOpen(false)}
  title="Panel title"
  size="md">
  Panel content
</SidePanel>;
```

## Slide over

The default variant slides over the page content from the right (or left with
`placement="left"`). Add `includeOverlay` to render a backdrop overlay.

## Slide in

Set `slideIn` and provide a `selectorPageContent` CSS selector pointing to the
page content element. The panel will push that element aside instead of
overlapping it.
